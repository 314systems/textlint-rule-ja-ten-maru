# textlint-rule-ja-ten-maru

日本語テキストの読点や句点を `、`/`。` に統一する
[textlint](https://textlint.org/) ルールです。

全角コンマ・全角ピリオドや、半角の読点・句点が混入していないかを検出し、自動修正（`--fix`）にも対応します。

## 変換ルール

| 検出する文字        | コードポイント | 変換後      | コードポイント | 分類          |
| ------------------- | -------------- | ----------- | -------------- | ------------- |
| `，` (全角コンマ)   | U+FF0C         | `、` (読点) | U+3001         | 読点 (`ten`)  |
| `､` (半角読点)      | U+FF64         | `、` (読点) | U+3001         | 読点 (`ten`)  |
| `．` (全角ピリオド) | U+FF0E         | `。` (句点) | U+3002         | 句点 (`maru`) |
| `｡` (半角句点)      | U+FF61         | `。` (句点) | U+3002         | 句点 (`maru`) |

## インストール

```sh
npm install -D textlint-rule-ja-ten-maru
```

## 使い方

`.textlintrc` でルールを有効化します。

```json
{
	"rules": {
		"ja-ten-maru": true
	}
}
```

### 実行例

```sh
# チェック
npx textlint README.md

# 自動修正
npx textlint --fix README.md
```

検出時には次の形式でメッセージが出力されます。

```text
Use "、" (U+3001) instead of "，" (U+FF0C).
```

## オプション

| オプション | 型        | デフォルト | 説明                         |
| ---------- | --------- | ---------- | ---------------------------- |
| `ten`      | `boolean` | `true`     | 読点 (`、`) への統一を行うか |
| `maru`     | `boolean` | `true`     | 句点 (`。`) への統一を行うか |

```json
{
	"rules": {
		"ja-ten-maru": {
			"ten": true,
			"maru": false
		}
	}
}
```

- `ten: false` — `，` `､` を検出・修正しません（句点のみ統一）。
- `maru: false` — `．` `｡` を検出・修正しません（読点のみ統一）。
- 両方 `false` の場合、何も報告されません。

## 例

```diff
-すべての人間は，生れながらにして自由であり､かつ，尊厳と権利とについて平等である．
+すべての人間は、生れながらにして自由であり、かつ、尊厳と権利とについて平等である。
```
